EasyZoom jQuery - image zoom plugin (modified)

(Check out this demo while hovering the image above)

EasyZoom is an basic, lightweight jQuery image zoom plugin based on the previous work by Alen Grakalic and Matt Hinchliffe. This modified version of EasyZoom includes a feature to show a preview of the magnified image. The original gallery extension and mobile device support has been removed.
